# join (조인)   

둘 이상의 테이블을 연결해서 데이터를 검색하는 방법   
테이블의 공통 컬럼을 통하여 테이블을 결합 한 후 데이터를 처리할 수 있다.   


~~~
종류
1. INNER JOIN   
교집합 공통적인 부분만 SELECT 된다.

2. LEFT / RIGHT JOIN   
조인조건의 기준에 따라 LEFT일 경우에는 왼쪽에 있는 기준 테이블 전부다 SELECT 된다.

3. OUTER JOIN    
두 테이블이 가지고 있는 전부다 SELECT 한다.


~~~
